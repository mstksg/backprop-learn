newtype DO (n :: Nat) Source #

Dropout layer. Parameterized by dropout percentage (should be between 0 and 1).

0 corresponds to no dropout, 1 corresponds to complete dropout of all nodes every time.

data StochFunc :: Maybe Type -> Type -> Type -> Type where Source #

Represents a random-valued function, with a possible trainable parameter.

Requires both a "deterministic" and a "stochastic" mode. The deterministic mode ideally should approximate some mean of the stochastic mode.
