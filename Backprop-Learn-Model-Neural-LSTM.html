data LSTM' (i :: Nat) (o :: Nat) Source #

Base for LSTM. An LSTM is just an LSTM' where the second half of the input vector is the previous output.

This is mostly an implementation detail. It is recommended that you use LSTM and its constructor, instead of directly using this.
