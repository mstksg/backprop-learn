    class (Num p, Num a, Num b) => Learn p a b l | l -> p, l -> a, l -> b where Source #

    Class for models that can be trained using gradient descent.

    An instance l of Learn p a b is parameterized by p, takes a as input, and returns b as outputs.

    l can be thought of as representing the hyperparameters of the model, with p representing the trained parameters of the model.

    If no trained parameters exist, NoParam can be used. This will automatically derive initParam.
