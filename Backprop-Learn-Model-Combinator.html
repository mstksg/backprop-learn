data Feedback :: Type -> Type -> Type -> Type where Source #

Take a model and turn it into a model that runs its output into itself several times, returning the final result. Parameterized by the number of repeats, and the function to process the output to become the next input.

See FeedbackTrace if you want to observe all of the intermediate outputs.

data FeedbackTrace :: Nat -> Type -> Type -> Type -> Type where Source #

Take a model and turn it into a model that runs its output into itself several times, and returns all of the intermediate outputs. Parameterized by the function to process the output to become the next input.

See Feedback if you only need the final result.

Compare also to Unroll.
